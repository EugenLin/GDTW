About us

本站「GDTW」（以下簡稱本網站）。

本站是一個基於網站前後端技術學習實踐的免費開源網站。預計順便提供一些免費且實用的小功能，短期目標是提供縮網址和圖片的上傳與瀏覽服務，中期可能視網站的完善程度將原始碼上傳GiHub做為開源專案共享。
本站目前規劃使用技術如下：
    Java
    SpringBoot
    MySQL
    Redis
    Ubuntu
本站軟硬體設備如下：
    Intel i5-12400
    Asrock Deskmini B760
    Teamgroup Elite Laptop DDR4 3200MHz 16GB*2
    Transcend SSD MTE220S 2TB*2
    Ubuntu 22.04 LTS
    CHT HiNet 500M/500M
本網站營運方式：
    自行搭建的Ubuntu伺服器，自行出資購買硬體和負擔電費與網路費等必要支出。
    本網站並無計畫導入外部廣告或提供商業加值服務，本網站將堅持提供免費服務。
    本站已盡量只使用網路上標註免費公開使用的素材，如仍有誤用或疑義請來信站方，本站會盡快更換素材。
    若使用者支持本站理念，可在本站日後開啟贊助服務時，視個人能力自願贊助本站營運費用。
